Ace-teroids
===========

A 2D space shooter in the style of Asteroids. Destroy asteroids, fight enemy
ships, and take down the boss across three stages.

Play in your browser (no install needed):
https://frankhurrican.github.io/Ace-teroids/

Download and run locally:
  - Windows: run bin/Ace-teroids.exe
  - Any platform: install Love2D 11.x from https://love2d.org/ then run:
      love Ace-teroids.love

Controls:
  WASD        - Thrust (momentum is maintained — no brakes!)
  Left click  - Shoot (fire up to 3 shots in quick succession)
